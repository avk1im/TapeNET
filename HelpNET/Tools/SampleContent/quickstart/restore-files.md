# Restoring files

1. Open the tape drive in the tree view on the left.
2. Select the backup set that contains the files you need.
3. Click **Restore** in the toolbar.
4. In the Restore dialog, choose a destination folder.
5. Optionally filter which files to restore using the filter pane.
6. Click **Start restore**.

Restored files are written to the destination folder, preserving their original
relative paths.
