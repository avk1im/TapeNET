# Your first backup

1. Insert a blank tape into the drive.
2. Click **Backup** in the toolbar.
3. Choose the folders you want to protect.
4. Optionally add an FCL filter to exclude temporary files.
5. Click **Start backup**.

TapeWinNET writes a new backup set to the tape. Progress is shown in the log
pane at the bottom of the window.
