# Welcome to TapeWinNET Help

TapeWinNET is a tape backup utility for Windows. Use the navigation tree on the
left to browse topics, or type a question in the chat pane at the bottom.

Key features include:

- **Full and incremental backups** to tape drives.
- **Flexible file filtering** via the FCL (File Conditions Language).
- **Restore and verify** workflows with progress reporting.
- **AI-assisted search** — ask questions in plain English.
