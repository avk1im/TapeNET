## [Backup sets list] Select a backup set

In the main window, navigate to a tape in the **tree view**, then look at the
**backup sets list** in the centre pane.

Tick the **[backup set](help://glossary/backup-set)** that contains the files you want to
recover.  You can tick multiple sets if the files span more than one incremental in the chain.

## [action:restore] Open the Restore dialog

Once you have selected the sets, open the **Restore** dialog:

- Click **[Open Restore…](help://action/restore)** right here, or
- Use the **Restore → Restore Backup Set…** menu.

The dialog has its own **Guide Me** tour that continues from here.
