# Welcome to TapeWin Help

TapeWin is a tape backup manager for Windows.  Use the sections below to
find what you need, or **type a question** in the chat box at the bottom of
this pane.

## Getting Started

- [Your first backup](help://topic/quickstart.first-backup) — back up files to tape in minutes.
- [Restoring files](help://topic/quickstart.restore-files) — retrieve files from a backup set.

## Core Concepts

- [Backup sets](help://topic/concepts.backup-sets) — how TapeWin organises data on tape.
- [FCL file filters](help://topic/concepts.fcl-filters) — powerful file-selection language.
- [Incremental backup](help://topic/concepts.incremental-backup) — back up only what changed.
- [Restore, Validate, Verify](help://topic/concepts.restore-validate-verify) — three ways to check your backups.

## Dialogs

- [New Backup dialog](help://topic/dialog.backup) — options for creating a backup set.
- [Restore dialog](help://topic/dialog.restore) — options for restoring files.
