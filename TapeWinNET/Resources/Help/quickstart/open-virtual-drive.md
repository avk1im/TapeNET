# Open a Virtual Drive

No tape hardware?  A **virtual drive** lets you try every TapeWin workflow using
a file (or RAM) as the tape.

## Steps

1. Choose **File → Open Virtual Drive…**.
2. Select **Create new virtual media** (or **In-memory** for a quick,
   throw-away test).
3. Optionally pick a **Preset** and click **Apply** to fill in capacity and
   features, then adjust if needed.
4. For file-backed media, choose a **content file** path.
5. Click **Create** (or **Open**).

The virtual drive appears in the tree, ready to
[format](help://topic/dialog.format-media) and
[back up to](help://topic/quickstart.first-backup).

See the [Open Virtual Drive dialog](help://topic/dialog.open-virtual-drive) for
every option, and [Virtual drives](help://topic/concepts.virtual-drives) for the
concept.
