# Status Bar

The status bar along the bottom of the window summarises the current state:

- **Drive / media status** — whether a drive is open and media is loaded.
- **Remote connection** — when connected to a remote host, a segment shows
  *Remote: host:port* (see [Remote service](help://topic/concepts.remote-service)).
- **Capacity / usage** — a quick read on used versus remaining space, also
  available as a usage bar via **View → Show Usage Bar**.
