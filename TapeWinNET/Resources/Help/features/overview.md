# Features Overview

TapeWin is a full-featured tape backup manager.  At a glance:

- **Backup** — full and [incremental](help://topic/features.incremental) backups,
  append or overwrite, with rich [file filtering](help://topic/features.fcl-filtering).
- **Restore / Validate / Verify** — recover files, or check integrity without
  writing — see [Restore, Validate, Verify](help://topic/concepts.restore-validate-verify).
- **[Multi-volume](help://topic/features.multi-volume)** — span large backups
  across several tapes.
- **[Virtual drives](help://topic/features.virtual-drives)** — work without
  hardware using file- or RAM-backed media.
- **[Remote host](help://topic/features.remote-host)** — operate a tape drive on
  another machine over the network.
- **Logging** — a filterable, exportable log pane that records every operation.
