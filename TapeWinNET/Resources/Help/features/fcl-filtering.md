# FCL Filtering

The **File Conditions Language (FCL)** is a compact language for selecting
files by name, path, extension, size, date, and attributes.  Use it to control
exactly what goes into a backup or comes out of a restore.

- **Pattern mode** — quick DOS-style wildcards (`*.docx; *.xlsx`).
- **Advanced mode** — a full FCL expression built in the
  [Advanced Filter editor](help://topic/dialog.fcl-filter).

See [FCL file filters](help://topic/concepts.fcl-filters) for the language
reference.
