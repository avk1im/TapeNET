# Remote Host

TapeWin can operate a tape drive — physical or virtual — attached to **another
machine** over the network, optionally secured with TLS.  Connect with
[Connect to Remote Host](help://topic/dialog.connect-remote-host), then mount a
drive from the **Drive** menu or open a
[remote virtual drive](help://topic/dialog.open-remote-virtual-drive).

See [Remote service](help://topic/concepts.remote-service) for how it works.
